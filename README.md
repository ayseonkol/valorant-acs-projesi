# Valorant İstatistik Analiz Projesi

Bu çalışma, Valorant oyuncu verilerini kullanarak istatistikler arasındaki ilişkileri inceleyen ve Python üzerinden bu verileri anlamlandıran bir Flask web uygulamasıdır.

## 🚀 Proje İçeriği
- **Veri Modelleme:** KDA, ADR ve Kast gibi teknik verilerin işlenmesi.
- **Web Paneli:** Kullanıcının değerlerini girip sonuçları görebildiği interaktif bir arayüz.
- **Veri Seti:** Analiz için kullanılan `players_stats.csv` dosyası.

## 🛠️ Teknik Detaylar
- **Dil:** Python
- **Kütüphaneler:** Flask, Scikit-learn, Pandas
- **Arayüz:** HTML (Templates)
